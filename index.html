1. Canvas properties for drawing

  1. fillStyle: style to use when filling objects; property values: css color, gradient, pattern; defaults to black
  2. strokeStyle: style to use on object strokes; properties are same with fillStyle
  3. lineWidth: width og the pen to use when drawing lines; defaults to 1
  4. 2 STEPS: first; set the fill and stroke styles. then; call drawing operations

2. Rectangles

  1. Rectangles are the only primitive shape supported by canvas
  2. clearRect(x, y, w, h) : Erases the given rectangle on canvas
  3. strokeRect(x, y, w, h) : Strokes the specified rectangle with current stroke style
  4. fillRect(x, y, w, h) : Fills the specified rectangle with current fill style



3. Lines

  1. moveTo(x, y) : Moves the drawing pen to x,y
  2. lineTo(x, y) : Draws a line from current pos to x,y
  3. beginPath() : Begins a set of multiple drawing commands as a single path
  4. stroke() : Strokes the currently open path that was started by beginPath



4. State

  1. save() : Saves the current drawing context
  2. restore() : Restores the most recent saved drawing context and makes it current



5. Arcs and Paths

  1. path : a set of connected points, and either is open or closed
  2. The drawing context always has one and only one current path
  3. beginPath(), closePath()
  4. arc(x, y, radius, startAngle, endAngle [, anticlockwise]) : curves that are portions of a circle == circle is a full arc



6. Curves

  1. bezier : is defined by 4 points: start, end, and two control points to determine how the curve is drawing from start to end
  2. bezierCurveTo(cx1, cy1, cx2, cy2, end1, end2) : Draw a bezier curve starting at the current pen position using the control points (cx1, cy1) and (cx2, cy2) and ending at (end1, end2)
  3. quadratic : is defined by 3 points: start, end, and one control point
  4. quadraticCurveTo(cx, cy, x, y) : Draw a quadratic curve starting at the current pen position using the control point (cx, cy) and ending at (x, y)



7. Texts

  1. fillText(str, x, y [, maxW]) : Renders the text at (x, y) but no wider than maxW
  2. strokeText(str, x, y [, maxW]) : Renders the text at (x, y) but no wider than maxW
  3. measureText(str) : Returns a TextMetrics object containing width of str
  4. font, textAlign, textBaseline


8. Shadows

  1. shadowColor, shadowOffsetX, shadowOffsetY, shadowBlur


9. Pattern

  1. ctx.createPattern(elem, type) : elem can be image, video, other canvas
  2. ctx.fillStyle = pattern

10. Gradient

  1. linear, radial
  2. ctx.createLinearGradient(x0, y0, x1, y1) : Creates a linear gradient using the two points to indicate the direction
  3. ctx.createRadialGradient(x0, y0, r0, x1, y1, r1) : Creates a radial gradient
  4. gradient.addColorStop(position, color) : Adds a color stop to the gradient; position should be between 0 and 1, color should be a css-style color
  5. Create gradient --> Add color stops --> ctx.fillStyle = gradient --> Create a shape

11. Images and Video

  1. drawImage(src, x, y) : Draws the given source image at (x, y)
  2. drawImage(src, x, y, w, h) : Draws the given source image at (x, y)
    and scales it to fit the given width and height
  3. drawImage(src, sx, sy, sw, sh, dx, dy, dw, dy) : Draws the given source image at (x, y)
    gets source image data from (sx, sy) and within the width and height given by sw and sh
    and draws he image on the destination at (dx, dy) and scales it to fit the given width and height dw and dy
  4. VIDEO is just drawImage with requestAnimationFrame



12. Clipping Path

  1. A clipping path is a type of mask
  2. Initially, the entire canvas is the default clipping path
  3. Any path can be a clipping path
  4. clip()


13. Transforms & Opacity & Compositing

  1. Controls how drawing is performed
  2. Any path can be a clipping path
  3. translate(x, y)
  4. scale(x, y)
  5. rotate(angle) : For rotate from a specific point: first translate, then rotate, then translate back
  6. ctx.globalAlpha = 0.4 (0 to 1)
  7. ctx.globalCompositeOperation : source-over, source-in, source-out, source-atop, lighter, xor, destination-over, destination-in, destination-out, destination-atop, darker, copy


14. Manipulating Raw Pixels

  1. Pixels can be manipulated and put back into canvas
  2. Each row of an image is composed of 4-byte pixels: RED, GREEN, BLUE, ALPHA
  3. WARNING: SAME ORIGIN
  4. imgData.data
  5. createImageData(w, h)
  6. createImageData(srcImg)
  7. getImageData(x, y, w, h)
  8. putImageData(imgData, x, y [, rect])
